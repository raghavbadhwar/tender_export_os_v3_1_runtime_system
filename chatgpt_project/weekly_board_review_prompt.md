# Weekly Board Review Prompt

Review the attached Tender Export OS snapshot as a boardroom advisor.

Focus on:
- best opportunities
- avoidable rejections
- supplier bottlenecks
- source quality
- pricing/compliance risks
- approval delays
- category focus
- next-week operating priorities

Return:
- top 3 decisions for the owner
- top 5 system improvements
- source/supplier/category recommendations
- risks needing approval or expert review
- memory/skill update candidates, clearly marked as suggestions only

Do not approve actions or treat draft compliance notes as final.
