# Category Strategy Review Prompt

Given the snapshot and any provided case reports, rank the current categories by practical operator fit.

Criteria:
- supplier availability
- margin potential
- compliance difficulty
- capital requirement
- deadline fit
- repeatability
- buyer/source quality
- founder learning value

Return:
- ranked categories
- why each is attractive or risky
- first 3 sources to monitor
- first 3 supplier segments to build
- one category to avoid for now
- evidence gaps
