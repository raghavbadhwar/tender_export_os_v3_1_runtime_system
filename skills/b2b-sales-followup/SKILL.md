# B2B Sales Follow-up

Use this skill for internal buyer or supplier follow-up drafts.

## Inputs
- `case_id`
- case report
- supplier/buyer context
- approval status
- prior correspondence or notes

## Procedure
1. Confirm whether the message is internal draft or approved external send.
2. Draft concise, factual copy with no unverifiable personalization.
3. Include exact case/product context and requested next step.
4. If external send is proposed, create or link an approval card.
5. Append run log when the draft is produced.

## Must Not
- send email/message without approval
- invent buyer/supplier facts
- commit price, delivery, payment terms, HSN/ITC-HS, or origin
- use generic inboxes when a verified direct recipient is required
