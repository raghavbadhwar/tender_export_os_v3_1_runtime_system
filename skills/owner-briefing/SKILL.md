# Owner Briefing

Use this skill for daily Hermes owner briefs.

## Required Format
1. New opportunities scanned
2. Auto-rejected
3. Best opportunities
4. Pending supplier proof
5. Approval required
6. Risks and blockers
7. Recommended owner action

## Procedure
1. Read active cases, run log, approval register, source health, and plugin health.
2. Show only decision-useful information.
3. Include `case_id` on every case-specific item.
4. Rank by score, deadline, and approval urgency.
5. End with one smallest useful owner action.

## Must Not
- include raw tender PDFs
- dump unscored leads
- fabricate status
- forward externally unless owner/gateway setup explicitly requires it
