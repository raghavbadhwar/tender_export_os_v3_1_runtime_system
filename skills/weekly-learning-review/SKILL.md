# Weekly Learning Review

Use this skill for Friday learning reviews.

## Inputs
- closed/won/lost/rejected cases
- source health
- supplier performance
- pricing misses
- owner corrections
- approval decisions
- plugin health

## Procedure
1. Summarize wins, losses, avoidable rejections, and blockers.
2. Identify source, supplier, pricing, compliance, and workflow lessons.
3. Stage memory updates only for durable lessons.
4. Stage skill/rule patches only when reusable.
5. Create approval cards for pricing/compliance skill changes if needed.
6. Generate a ChatGPT snapshot for strategic review when useful.

## Must Not
- store raw tenders/RFQs/supplier tables in memory
- patch skills silently
- treat ChatGPT output as final compliance authority
